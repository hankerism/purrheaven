# Purr Heaven 🐾

A cozy cat adoption and cat café website designed to help rescue cats find their forever homes.

## Features

- Cat adoption listings
- Online visit reservations
- Donation page
- Cat gallery slideshow
- Success stories
- Responsive design

## Pages

- Home
- About
- Meet Our Cats
- Adoption
- Visit Us
- Gallery
- Donations
- Contact

## Mission

At Purr Heaven, we believe every purr deserves a home. Our goal is to connect loving families with rescue cats while creating a welcoming community for cat lovers.
